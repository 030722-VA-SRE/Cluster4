How to use Chip-Shop API

* Post: auth/login
    * for user login
* Post: auth/register
    * for user to register a new account
    * registers users are only customer accounts
* GET: /brands
    * retrieve all brands
* GET: /flavors
    * retrieve all flavors
* GET: /brands/1/flavors
    * retrieve all flavors of that brand
